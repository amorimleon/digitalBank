## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone git@github.com:amorimleon/digitalBank.git
```

No terminal do VsCode, Instale o Express:

```
npm install express
```

Para executar o express use o seguinte comando:
```
npm run dev
```
